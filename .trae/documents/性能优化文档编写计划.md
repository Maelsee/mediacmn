# 文档结构
- 概述：优化目标、范围与背景
- 已实施优化：刮削会话复用/聚合、侧车并发上传与限额、日志压缩
- 配置项：`.env` 或环境变量说明（SIDE_CAR_LOCALIZATION_ARTWORK_LIMIT、SIDE_CAR_LOCALIZATION_ENABLED、TASK_EXECUTOR_COUNT 等）
- 运行指标：优化前后对比（来自日志样例：3.58s/4.79–5.91s），统计口径与观测方法
- 操作指南：如何启用/调整并发与限额、如何查看上传耗时与失败计数、执行器并发建议
- 日志策略：仅在任务完成输出 info，其余降级为 debug 的清单
- 未来工作：详情聚合完全落地、缓存与限流/熔断联动、更多并发调优
