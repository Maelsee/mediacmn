# 目标
- 将统一执行器与统一调度器的非关键日志降级为 debug，仅在任务完成（成功/失败）时保留 info。

# 范围
- `services/scan/unified_task_executor.py`
  - 初始化、启动、停止请求、停止、开始执行任务：info → debug
  - 保留：任务执行成功（info）、失败（warning/error）
- `services/scan/unified_task_scheduler.py`
  - 初始化完成、创建任务相关、执行开始、执行完成、组合任务中间信息、删除对齐完成：info → debug
  - 保留：错误（error）、权限警告（warning）

# 验收
- 正常运行时，仅在“任务完成”处出现 info 日志（来自执行器和队列服务）；其他流程性日志降为 debug，不再噪音。

# 回滚
- 如需恢复详细信息，将对应 debug 恢复为 info 或增加 env 开关控制日志级别。
